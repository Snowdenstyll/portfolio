## Overview

Coup-Me is a **coupon aggregator platform** designed to simplify the way users discover and redeem discounts. Built with a focus on usability and scalability, Coup-Me enables users to log in effortlessly, track their savings, and find relevant deals through a robust and intuitive interface.\
Visit [coup-me.com](https://coup-me.com/)


## Key Features

- **OAuth Integration**: Implemented Google and Facebook OAuth APIs to provide secure, one-click user authentication, enhancing login efficiency.
- **Real-Time Analytics**: Integrated Google Analytics to track user behavior and generate actionable insights for improving the platform.
- **Scalable Infrastructure**: Provisioned and managed **AWS EC2 servers** to establish distinct production and development environments, ensuring reliable performance.
- **Responsive Design**: Styled the platform using **HTML, CSS, and JavaScript** for a seamless user experience across all devices.

## Technologies Used

- **Laravel**: For backend development and efficient coupon aggregation.
- **jQuery**: For dynamic front-end interactions.
- **Google Analytics**: To monitor user activity and optimize the platform.
- **AWS EC2**: For hosting scalable and secure environments.

## Challenges and Learnings

One challenge was integrating multiple OAuth providers while maintaining a consistent and secure user authentication experience. Additionally, provisioning AWS environments required meticulous planning to optimize costs without sacrificing performance. These efforts honed expertise in infrastructure management and API integrations.

## Outcome

Coup-Me has grown into a reliable platform for coupon enthusiasts, offering seamless authentication, user-focused design, and robust performance. The project underscores the value of integrating scalable technologies with data-driven insights to enhance user satisfaction and engagement.
